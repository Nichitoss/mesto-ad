## Mesto (API + Vite)

Проект Mesto с подключением к API `mesto.nomoreparties.co`, улучшенным UX форм и сборкой через Vite.

### Ссылка на проект

- **GitHub Pages**: добавьте ссылку после публикации

### Установка и запуск

```bash
npm install
npm run dev
```

### Сборка и публикация

```bash
npm run build
npm run deploy
```

### Настройка API

В файле `src/scripts/components/api.js` укажите:

- **cohortId**: идентификатор группы
- **token**: личный токен
